# GitHub Push Instructions

## After Creating Your Repository on GitHub

Once you've created a new repository on GitHub (let's assume it's called `catena-aurea-project`), run these commands:

```powershell
# Add GitHub as remote origin (replace 'yourusername' with your actual GitHub username)
git remote add origin https://github.com/yourusername/catena-aurea-project.git

# Push to GitHub
git push -u origin main
```
